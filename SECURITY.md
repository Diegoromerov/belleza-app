# Security Policy

## Reporting a Vulnerability

We take the security of GlowApp seriously. If you find a security vulnerability, please do NOT create a public issue. Instead, report it privately.

Please send security reports directly via email to: **security@beautyapp.com**

Please include the following information in your report:
- Type of issue (e.g., buffer overflow, SQL injection, cross-site scripting)
- Steps to reproduce the issue (or a proof-of-concept script/exploit)
- Potential impact of the vulnerability

We will acknowledge receipt of your report within 48 hours and work with you to resolve the issue promptly.
